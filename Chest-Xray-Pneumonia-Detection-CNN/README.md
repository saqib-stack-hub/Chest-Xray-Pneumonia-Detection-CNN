# 🩺 Chest X-ray Pneumonia Detection using CNN

## 📌 Project Overview

This project uses a Convolutional Neural Network (CNN) to classify Chest X-ray images into Normal and Pneumonia categories. The model was developed using TensorFlow/Keras and trained in Google Colab. The objective is to demonstrate how Deep Learning can be applied to medical image classification.

---

## 🚀 Features

- Chest X-ray image classification
- CNN-based Deep Learning model
- Image preprocessing and normalization
- Training and validation accuracy visualization
- Training and validation loss visualization
- Ready for Streamlit deployment

## 🧠 CNN Architecture

- Conv2D (32 Filters)
- MaxPooling2D
- Conv2D (64 Filters)
- MaxPooling2D
- Conv2D (128 Filters)
- MaxPooling2D
- Flatten
- Dense (128)
- Dropout (0.5)
- Output Layer (Softmax)

The CNN model achieved high training and validation accuracy while maintaining low validation loss, indicating good learning performance.
